createBlockChannelWriter的使用分析

Posted on 2019-10-07 In 学习

其实这篇文章应该叫做blockchannel的读写分析更恰当。

Basics

file channel

首先我们从file channel开始介绍。
FileChannel类是Java的NIO库里面的，FileChannel和Bytebuffer结合进行读写数据。
它的优点有：

  • 可以在特定位置读写文件
  • 把文件的一部分放在内存中，更高效
  • 可以把文件数据从一个channel传输到另一个，并且更快
  • 可以锁住一个文件区域的数据，放置其他线程访问
  • 为了防止数据丢失，可以强制写文件更新到存储

    LinkedBlockingQueue

AsynchronousBlockWriter

在iomanagerasync.java中，createBlockChannelWriter的实现就是创建了一个AsynchronousBlockWriter。
AsynchronousBlockWriter的作用，根据注释，是为给定的File channel创建一个新的块channel writer。
